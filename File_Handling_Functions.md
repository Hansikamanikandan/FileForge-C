## 📚 File Handling Functions Used

| Function | Purpose |
|----------|---------|
| `fopen()` | Opens a file in the specified mode (Read, Write, Append, etc.) |
| `fclose()` | Closes an opened file and releases resources. |
| `fgetc()` | Reads a single character from a file. |
| `fputc()` | Writes a single character to a file. |
| `fprintf()` | Writes formatted output to a file. |
| `fscanf()` | Reads formatted input from a file. |
| `EOF` | Indicates the End Of File while reading. |

### File Modes

| Mode | Description |
|------|-------------|
| `r` | Open an existing file for reading. |
| `w` | Create a new file or overwrite an existing file. |
| `a` | Append data to the end of a file. |
| `r+` | Read and write an existing file. |
| `w+` | Read and write (creates/overwrites file). |
| `a+` | Read and append data. |
